# Sea_Animal_Classification
This project focuses on building an image classification system to identify different types of sea animals using deep learning. The model is developed using the ResNet50 architecture, which is a powerful pre-trained convolutional neural network commonly used for image recognition tasks. The dataset used in this project contains a total of 13,712 images belonging to 23 different sea animal classes. The dataset was divided into three parts: 10,970 images were used for training, 1,371 images for validation, and 1,371 images for testing the model. Before training, all images were preprocessed and resized to 224 × 224 pixels to match the input size required by ResNet50. The data was organized into train_data, validation_data, and test_data folders, and a batch size of 32 was used during training. The model was trained for 50 epochs, allowing it to learn important visual features of different sea animals from the images. After training, the model achieved an accuracy of 0.9149 (91.49%), showing that it performs well in correctly classifying sea animal images. This project demonstrates how transfer learning with ResNet50 can be effectively used for marine species classification and can be useful in applications such as marine research, biodiversity monitoring, and educational tools.
